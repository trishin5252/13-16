# Заметки - PWA Приложение с Service Worker

Офлайн-приложение для управления заметками с использованием Service Worker для кэширования ресурсов.

## Выполненные требования

### Интерфейс (WPF)
- [x] Список сетевых интерфейсов компьютера
- [x] Информация о выбранном интерфейсе
- [x] Поле для проверки и анализа URL/URI
- [x] Область отображения результатов

### Функционал для работы с IP
- [x] Получение списка всех сетевых интерфейсов (System.Net.NetworkInformation)
- [x] IP-адрес
- [x] Маска подсети
- [x] MAC-адрес
- [x] Состояние подключения
- [x] Скорость соединения
- [x] Тип интерфейса (Ethernet, Wi-Fi, Loopback)

- [x] Схема (протокол)
- [x] Хост
- [x] Порт
- [x] Путь
- [x] Параметры запроса
- [x] Фрагмент

- [x] Получение DNS-информации
